•PWM Signal Generator
This project presents the design and simulation of a PWM (Pulse Width Modulation) signal generator using OrCAD.

•Project Description
The circuit generates a PWM signal by comparing a reference signal with a periodic waveform using a comparator. The design allows adjustment of the duty cycle and output amplitude.

 •Features
- Adjustable duty cycle
- Adjustable signal amplitude
- Comparator-based PWM modulation

• Simulation and Analysis
The circuit behavior was evaluated through several analyses:
- Time-domain analysis
- Parametric analysis
- Monte Carlo analysis
- Worst-case analysis

• Tools Used
- OrCAD Capture
- PSpice
